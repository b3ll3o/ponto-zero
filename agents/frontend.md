# @frontend

## Profile

- **Role**: UI
- **Specialty**: Next.js, mobile-first, a11y, pwa
- **Agent**: opencode (or other frontend agent)

## Skills
- nextjs-16
- mobile-first
- a11y
- pwa
- typescript
- tailwind

## Responsibilities

1. Implementar interfaces conforme specs
2. Garantir mobile-first design (320px+)
3. Implementar acessibilidade (WCAG 2.1 AA)
4. Implementar PWA features
5. Escrever testes unitários de componentes
6. Escrever testes e2e de fluxos UI

## Deliverables

- Componentes React: `src/components/`
- Páginas: `src/app/`
- Testes: `*.test.tsx`, `e2e/*.spec.ts`

## Acceptance Criteria

- Mobile-first design
- WCAG 2.1 AA compliance
- Lighthouse PWA > 90
- Coverage > 80%

## Boundaries

- **DO**: UI, components, pages, styling
- **DON'T**: API routes, database schemas, auth logic
